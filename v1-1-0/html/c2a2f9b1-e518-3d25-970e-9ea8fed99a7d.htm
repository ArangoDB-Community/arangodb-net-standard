Properties
  NameDescription
Level
Optional. Retrieves all log entries of this log level. Note that the query parameters UpTo and Level are mutually exclusive.
Offset
Optional. Specifies the number of records to skip when retrieving log entries. Offset and Size can be used for pagination.
Search
Optional. Retrieves all log entries containing this text string.
ServerId
Optional. Retrieves all log entries of the specified server. All other query parameters remain valid. If no serverId is given, the asked server will reply. This parameter is only meaningful on Coordinators.
Size
Optional. Specifies the number of log entries to retrieve.
Sort
Optional. Specify whether to sort log entries in ascending or descending order. Only SortAscending or SortDescending are accepted here. Log entries are sorted by their Id values. The default value is SortAscending.
Start
Optional. Retrieves all log entries that their log entry identifier (lid value) is greater or equal to this value.
UpTo
Optional. Retrieves all log entries up to this log level.
